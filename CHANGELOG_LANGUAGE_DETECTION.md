# Multi-Language TTS Guide

Quick guide for generating speech in different languages using Edge TTS and Piper TTS.

---

## Edge TTS (Online)

### Step 1: Enter Your Text
Enter or import text in your desired language (e.g., French text for French speech).

### Step 2: Choose Language
Click the language dropdown to open the language selector.

### Step 3: Filter Languages
Start typing the language name to filter (e.g., type "French").

### Step 4: Select Language & Voice
Select your language (e.g., French CA), then choose a voice from the dropdown.

### Step 5: Generate
Click **Generate MP3** to create your audio file.

---

## Piper TTS (Offline)

### Step 1: Language Detection
When you select a language without available voices, a prompt will appear asking to download voices.

### Step 2: Download Voices
Click **Download voices** to open the Download Manager. Select individual voices or click **Download All**.

### Step 3: Wait for Download
Wait for voice files to download, then close the popup.

### Step 4: Generate
Select your downloaded voice and click **Generate WAV**.

---

## Key Features

| Feature | Edge TTS | Piper TTS |
|---------|----------|-----------|
| Connection | Online (Microsoft) | Offline |
| Output Format | MP3 | WAV |
| Voice Download | Not needed | Required per language |
| Auto Language Detection | Yes | Yes |

---

## Tips

- **Auto-filter**: Type text and voices auto-filter by detected language
- **Show All**: Check "Show all languages" to see voices grouped by language
- **New Languages**: Piper prompts you to download voices when a new language is detected
